Manual de marca ZN. — sistema de wordmark, paleta, tipografia e aplicações.

Este documento define o uso correto da marca ZN. nos canais digitais e impressos do ecossistema Thiago Zeni. Quatro variantes do wordmark, mais duas versões mono, compõem o sistema. Tudo aqui é regra, não sugestão.

01 — identidade

O que é ZN.

ZN. é o monograma de Thiago Zeni — wordmark sólido, brutalista, com um único accent lime (o ponto). Funciona como assinatura pessoal-executiva: autoridade sem ostentação, contemporâneo sem cair no genérico tech. O ponto não é decoração — é o pulse da marca, único elemento que pode se mover, brilhar ou variar de cor controladamente.

02 — sistema de variantes

Quatro expressões + duas mono. Mesma marca.

Toda variante mantém: Unbounded 900, tracking −0.08em, ponto presente. O que muda é tratamento de superfície (sólido, recorte, outline, ruptura) e contexto de uso. As duas mono cobrem restrições reais (impressão a 1 cor, fax, bordado, gravação). Master = Solid. Tudo deriva dele.

03 — construção

Como o wordmark é construído.

O ZN. não é redesenhado a cada uso — é renderizado a partir da fonte Unbounded em peso 900, com o ponto manualmente colorido em #C6FF00. Isso garante consistência absoluta entre web, print e qualquer aplicação futura. Não vetorize manualmente. Não recrie. Use o SVG master ou renderize do CSS abaixo. 

.zn-mark {
  font-family: 'Unbounded', sans-serif;
  font-weight: 900;
  letter-spacing: -0.08em;
  line-height: 0.9;
  color: #F2F2F2;
}
.zn-mark .dot {
  color: #C6FF00;
  /* opcional · text-shadow: 0 0 28px rgba(198,255,0,0.55); */
}
<span class="zn-mark">ZN<span class="dot">.</span></span>
04 — espaço de respiro & tamanho mínimo

A marca respira.

Em volta do wordmark deve existir uma área livre de X = altura do "Z" em todas as direções. Nenhum elemento gráfico, texto ou imagem entra nessa zona. Em layouts apertados, prefira diminuir o logo a invadir o respiro.

05 — paleta cromática

06 — tipografia

Três famílias. Cada uma com seu papel.

Unbounded sustenta o wordmark e todos os títulos — é a espinha visual. DM Sans faz o trabalho do dia-a-dia (corpo de texto, navegação). Cormorant Garamond italic é o contraponto — citações, taglines, momentos de pausa. Nunca trocar Unbounded por Inter, SF Pro ou Roboto por "facilidade".

07 — superfícies & contraste

Onde a marca respira ou sufoca.

A marca foi construída pra fundo escuro — é onde o lime vibra mais. Em fundos claros funciona, desde que o lime tenha contraste suficiente com o texto. Fundos coloridos saturados são território perigoso — o lime perde força ou briga.

08 — proibições

Seis maneiras de quebrar a marca.

Estas são erradas. Sem exceção. Se a regra atrapalha o seu layout, o problema é o layout — não a marca. Pra cada "vontade" de variar, existe uma variante oficial (seção 02) que cobre o caso.

não
ZN.
Não distorça. Nunca alterar proporção horizontal/vertical. Use o tamanho diretamente — escala uniforme apenas.
não
ZN.
Não troque a cor. Lime é fixo. Recolorir a marca anula o sistema inteiro. Pra contextos sem lime, use Mono White ou Mono Black.
não
ZN.
Não rotacione. A marca é sempre horizontal. Sem inclinação, sem orientação vertical, sem "criatividade" angular.
não
ZN.
Não troque a fonte. Unbounded 900 é regra. DM Sans, Inter, Helvetica, Arial — todas anulam o ZN. Sem exceção, sem "no momento estou sem a fonte".
não
ZN.
Não abra o tracking. Letras separadas matam o peso brutalista. Tracking é fixo em −0.08em. Diminuir é OK em casos extremos, abrir nunca.
não
ZN.
Não aplique gradiente. A marca é cor sólida. Gradientes (mesmo lime → lime claro) viram ruído visual e quebram a hierarquia do ponto.
09 — aplicações reais
