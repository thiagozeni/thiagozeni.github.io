A tensão a resolver

Extremo A
ERPs verticais
Publi · Operand · iClips · Workamajig. Profundos, fiscalmente corretos, completos. Castigam o usuário com 80+ telas, formulários longos, sem IA.
Extremo B
Tools horizontais
Notion · Asana · Monday · ClickUp. Leves, rápidos, agradáveis. Decepcionam o gestor com superficialidade fiscal e zero compliance.
A tese
Combine profundidade vertical + leveza editorial.
O sistema deve sair do caminho. O trabalho não é navegar pelo software · é entregar o que o cliente pediu. A ousadia: ERP com a alma de Linear/Notion · não Notion com cara de ERP.

Visão · 5 decisões arquitetônicas
13

De 80+ telas para 25

Não cosmético. Estrutural. Cinco decisões eliminam dezenas de telas que o ERP tradicional acumularia.
01
Pessoa é entidade de primeira classe. "Hoje" · "Minha semana" · "Meus clientes" são as views default · não navegação por módulo.
02
Projeto é canvas único · não 11 abas. Página vertical fluida com timeline + tarefas + briefing + arquivos + financeiro + comentários · seções colapsáveis.
03
Mídia é plano visual · não planilha. Plano de Mídia É a timeline com PIs como blocos clicáveis. Drag = reagendar · click = drawer.
04
Financeiro é dashboard live · não relatórios. 4 KPIs + fluxo de caixa projetado + tabela filtravel. Exportar PDF é botão secundário, não o produto.
05
Cadastros viram dados sob demanda. 17 sub-tabelas hoje (acabamentos, bancos, cidades…) · ViaCEP automático · autocomplete inteligente · cria silenciosamente.

Regras de ouro · o que a IA NÃO faz
Não decide aprovações · não emite NF · não envia ao cliente sem revisão · não altera dados de mídia/financeiro automaticamente · não usa dados de cliente A para sugerir ao cliente B · não responde com achismo (diz "não sei, posso pesquisar"). IA é assistente, não substituto.

Visão · Sistema visual
25

Identidade com personalidade

Paleta
3 modos
Cinza papel quente · não branco puro. Verde = tempo/timer · laranja = CTA principal de fluxo · não se misturam. Modo escuro nativo · high-contrast acessível. Recusa o azul royal corporativo do Publi.
Tipografia
Inter ou Geist
Variável · suporta pt-br · hierarquia clara (Display · H1 · H2 · H3 · Body · Caption · Mono). JetBrains Mono para números. 40% do que faz software parecer cuidado.
Densidade
3 modos
Compacto 32px · padrão 44px · confortável 56px. Cmd+, alterna em tempo real. Power user fica em compact, novato em comfortable.
Motion
5 timings
Instant 0ms · snap 150ms · smooth 300ms · narrative 500ms · dramatic 800ms. Skeleton loaders sempre · spinner full-screen proibido.
Componentes
Drawer > modal
Drawer lateral 480px é default · modal full-screen é último recurso · podem se empilhar · Esc fecha o topo · Esc-Esc fecha tudo.
Acessibilidade
A11y nativa
Contraste WCAG AA mínimo · prefers-reduced-motion respeitado · keyboard-only navegável · screen reader em todas as ações principais.
